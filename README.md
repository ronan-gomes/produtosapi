# 📦 API de Produtos - Spring Boot

API REST simples desenvolvida com Spring Boot para **fins de estudo e aprendizado**.

---

## 🚀 Tecnologias

* Java
* Spring Boot
* Spring Data JPA
* Maven

---

## 📌 Funcionalidades

* CRUD de produtos
* Estrutura básica REST

---

## 📡 Endpoints

* `GET /produtos` → Lista todos os produtos
* `GET /produtos/{id}` → Busca por ID
* `POST /produtos` → Cria um novo produto
* `PUT /produtos/{id}` → Atualiza um produto
* `DELETE /produtos/{id}` → Remove um produto

---

## ▶️ Como executar

```bash
git clone https://github.com/ronan-gomes/produtosapi.git
cd produtosapi
./mvnw spring-boot:run
```

---

## 🎯 Objetivo

Projeto criado como **base de estudo**, para praticar desenvolvimento de APIs com Spring Boot.
